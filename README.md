## Install NeoVim in ArchLinux OS

in a arch linux run de follows command for install the **neovim**:

```shell
$ sudo pacman -S neovim
```

## Install Plugins

Open the **neovim**:

```shell
$ nvim
```

and execute **PlugInstall** command

```shell
:PlugInstall
```
